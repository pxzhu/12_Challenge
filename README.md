##### [Reddit Clone](#Reddit-Clone)

Reddit Clone
- 2020-10-20
>레일즈 프로젝트를 생성합니다.
```terminal
$ sudo rails new redditclone
```
>프로젝트 폴더로 이동합니다.    
>레일즈 서버를 시작해줍니다.    
``` terminal
$ cd redditclone
$ sudo rails server
```
>테스트를 완료하고 'Ctrl + C'를 이용해 레일즈 서버를 종료합니다.
```terminal
$ ^C
```
